# 📺 BrightTV Viewership Analytics  

## 📌 Executive Summary
BrightTV aims to grow its subscription base by leveraging user data. This project analyzes **viewership behavior, engagement patterns, and content performance** to deliver actionable insights for the CVM team.

---

## 🎯 Business Objective
- Increase subscription growth  
- Improve retention rates  
- Enhance user engagement  
- Optimize content and pricing strategies  

---

## 📊 Data Overview
**Data Sources:**
- User Profiles  
- Viewing Transactions  

**Key Fields:**
- Demographics (age, region, household)  
- Viewing activity (content, time, duration)  

---

## 🧹 Data Preparation
- Converted timestamps from **UTC → SAST**  
- Removed duplicates  
- Standardized missing values (`Unknown`)  
- Joined datasets into a unified analytical model using left join function 

---

## ⚙️ Feature Engineering
- Day of Week  
- Time of Day (Morning / Afternoon / Evening)  
- Session Duration Buckets  
- Content Categories  

---

## 📈 Key Insights

### 🔹 Engagement Trends
- Peak usage: **Evenings**  
- Highest activity: **Weekends (Saturday)**  
- Sports content → **High engagement**  
- Kids content → **Consistent usage**

### 🔹 Consumption Patterns
- Adults dominate evening traffic  
- Families engage more with kids & family content  


### 🔹 Low Consumption Signals
- Midweek drop (Mon–Wed)  
- Lower session counts linked to weak content engagement  
- Reduced retention where personalization is lacking  

---

## 🚀 Strategic Recommendations

### 📺 Content Strategy
